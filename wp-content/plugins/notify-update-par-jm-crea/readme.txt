== Description ==

Notify Update par JM Créa vous notifie par email et sms (pour les abonnés freemobile) lors d’une mise à jour de votre Wordpress.

Notify Update par JM Créa détecte les mises à jour de vos plugins, de vos thèmes et aussi de votre version de Wordpress et vous en informe par mail et par sms.

Le plugin est paramétrable au niveau de la planification, mais aussi vous pouvez choisir ou pas d’activer les notifications.

Si vous avez des questions, n’hésitez pas à en parler sur le support.

== Installation ==

1. Uploadez `Notify-Update-Par-JM-Crea` dans le dossier `/wp-content/plugins/` de votre site
1. Activez ’Notify Update’ depuis le menu « Extensions » de Wordpress
1. Rendez-vous ensuite sur ‘Outils’ dans votre menu à gauche puis cliquer sur ‘Notify Update’
